I had a very difficult time wrapping my head around this project without the hardware in my hands. Abstract development seems difficult to me. Once I felt more comfortable with the operational mechanisms I could confidently build individual components without so much hardware need.

I had problems with the rss feed, I tried to do the goeags goeags.rss which is a simple rss and it would not work, Then I tried an atom feed from a mental health link off the same search. nimh.atom The atom was successful. The issue as it was explained to me is that the eags feed calls into a namespace where the nimh feed uses an xml stylesheet. For some reason unknown to us because we cannot see inside this proprietary program it cannot read rss comming from a namespace.
